# WGSL Studio

VS Code extension for WGSL/WESL shader development: toy previews and testing.

## Features

- **Toy Shader Preview**: Live WebGPU preview for shaders with `@toy` annotation
- **Hot Reload**: Preview updates automatically on save
- **WESL Support**: Full support for imports, conditionals, and external packages (e.g., lygia)

## Usage

1. Open a `.wgsl` or `.wesl` file
2. Run command: `WGSL Studio: Preview Toy Shader`
3. Edit and save to see live updates
